# Source Directory

The first MVP is a static Azure Static Web Apps frontend at the repository root:

- `index.html`
- `styles.css`
- `app.js`

This `src` folder is reserved for a future framework-based implementation if the product outgrows the static MVP.

Keep the first implementation small and focused on the MVP:

- Ticket board
- Ticket form
- Dashboard summary
- Calendar view
- Delivery review flow
- AI-assisted ticket enrichment review step
